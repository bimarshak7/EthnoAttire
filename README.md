# EthnoAttire
**Eplore nepalese culture in smart ways** 
## Project Setup
```
git clone https://github.com/bimarshak7/EthnoAttire.git
cd EthnoAttire/client
npm install
cd ../server
pip install -r requirements.txt
```
## Run client
``` cd client && npm run dev```
## Run server
``` cd server && python index.py```
